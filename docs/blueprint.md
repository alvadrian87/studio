# **App Name**: EvoLadder Manager

## Core Features:

- Tournament Ladder Display: Display tournament ladder with player rankings and challenge status.
- Landing Page: landing page con informacion de todo lo que hace el sistema
- CRUD operations: tanto usuarios, torneos y partidos con metodologia CRUD
- Player Dashboard: Allow players to view their profile, challenge history, and current ranking y estadisticas del jugador(cant de sets ganados, porcentaje de games ganados y todo lo que se te pueda ocurrir)
- Challenge Issuance: Enable players to issue challenges to others on the ladder, subject to backend validation rules.
- Tournament Management: Allow the admin user to configure general settings, create new tournaments, update basic tournament settings. Use AI to validate tournament configurations. Uses AI to detect misconfigurations and suggests possible actions tool.
- Marketplace: marketplace para usuarios

## Style Guidelines:

- Primary color: Golden Yellow (#A37A36), reminiscent of trophies and success in a light mode.
- Background color: Very light grey (#F0EFF2), providing a clean and modern backdrop for content.
- Accent color: Analogous muted blue-gray (#778593), providing a cool, calming contrast to the warmth of the primary color.
- Body and headline font: 'Inter', a versatile sans-serif for a modern and clean user experience.
- Use simple, clear icons to represent actions and information, such as challenge status, player rankings, and tournament details. Preferably, tennis ball, ladder, shield and sword images and related sport action.
- Employ a clean, well-spaced layout to improve readability and overall user experience, with an emphasis on clear hierarchy.
- Use subtle animations for actions like issuing a challenge or updating rankings.
